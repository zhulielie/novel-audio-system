# 🧠 LLM小说网站智能分析指南

## 🎯 功能概述

LLM小说网站智能分析系统是一个基于AI的小说网站自动分析工具，能够：

- ✅ **智能识别** - 自动识别网站结构和内容模式
- ✅ **代码生成** - 自动生成适合该网站的爬虫代码
- ✅ **策略优化** - 提供最优的爬取策略建议
- ✅ **一键集成** - 直接在Django Admin中使用

#### 3. 置信度计算
```python
# 计算分析结果的置信度
def calculate_confidence(analysis_result):
    score = 0.0

    # 标题匹配度 (30%)
    if analysis_result.get('title'):
        score += 0.3

    # 章节数量 (20%)
    if len(analysis_result.get('chapters', [])) > 10:
        score += 0.2

    # 选择器准确性 (20%)
    if analysis_result.get('content_selectors'):
        score += 0.2

    # 网站特征匹配 (30%)
    if is_known_website(analysis_result.get('domain')):
        score += 0.3

    return min(score, 1.0)
```
