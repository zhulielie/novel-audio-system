🛡️ 安全爬虫使用指南
=====================================

安全设置已应用完成！

📊 当前配置:
• 每日请求限制: 30次
• 每小时请求限制: 12次
• 下载间隔: 12秒
• 页面加载等待: 18秒

🎯 使用方法:
1. 访问Django Admin: http://localhost:8000/admin/
2. 进入小说管理页面
3. 选择要下载的小说
4. 点击"🎭 Playwright自动下载"
5. 系统会自动应用安全设置并开始下载

⚠️  安全提醒:
• 每次下载前会检查请求限制
• 接近限制时会自动警告
• 超过限制会自动暂停
• 详细的日志记录所有操作

📈 监控状态:
• 今日请求: 0/30
• 小时请求: 0/12
• 剩余可用: 30次

如需调整设置，请运行:
python safe_crawler_config.py
