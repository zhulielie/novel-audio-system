# 📚 小说多来源爬取系统指南

## 🎯 系统概述

新的小说爬取系统采用了**多来源架构**，每个小说可以关联多个来源网站，系统会根据来源类型自动选择合适的爬取策略。

### 🏗️ 架构设计

```
小说 (Novel)
├── 基本信息 (标题、作者、简介等)
└── 多个来源 (NovelSourceRelation)
    ├── 来源网站 (NovelSource)
    │   ├── 类型: 和图书、起点、纵横等
    │   ├── 基础URL
    │   ├── 爬虫策略
    │   └── 统计信息
    └── 关联信息
        ├── 具体URL
        ├── 是否主来源
        └── 同步记录
```

## 🚀 快速开始

### 第一步：访问管理界面

访问以下URL管理不同类型的资源：
- **小说来源管理**: `http://localhost:8000/admin/novels/novelsource/`
- **小说关联管理**: `http://localhost:8000/admin/novels/novelsourcerelation/`
- **小说管理**: `http://localhost:8000/admin/novels/novel/`

### 第二步：创建来源网站

#### 添加新的来源网站
1. 进入"小说来源"管理页面
2. 点击"添加小说来源"
3. 填写信息：
   ```
   名称: 和图书
   来源类型: 和图书
   基础URL: https://www.hetushu.com
   章节URL模式: /book/{book_id}/{chapter_id}.html
   编码: utf-8
   优先级: 1 (数字越大优先级越高)
   ```

#### 支持的来源类型
- **和图书** (hetushu) - 主要支持
- **起点中文网** (qidian) - 实验性支持
- **纵横中文网** (zongheng) - 实验性支持
- **自定义** (custom) - 可扩展

### 第三步：关联小说与来源

#### 方式1：通过小说管理
1. 进入小说编辑页面
2. 在"来源管理"部分选择来源
3. 输入具体的URL（如：`https://www.hetushu.com/book/38/24721.html`）
4. 设置是否为主来源

#### 方式2：通过关联管理
1. 进入"小说来源关联"页面
2. 点击"添加小说来源关联"
3. 选择小说和来源
4. 输入具体URL

### 自定义爬取逻辑

#### 重写解析方法
```python
class CustomCrawler(BaseCrawler):
    def parse_chapter_list(self, html_content):
        """自定义章节列表解析"""
        soup = BeautifulSoup(html_content, 'html.parser')

        # 自定义解析逻辑
        chapters = []
        for link in soup.find_all('a', class_='custom-chapter-link'):
            chapters.append({
                'title': link.get_text().strip(),
                'url': urljoin(self.source_url, link['href']),
            })

        return chapters
```
